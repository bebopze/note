1、书

    我前期把博士的所有文章都反复读过很多遍，甚至去翻了他十年前的博客，还有其他的一些师兄的文章。

    包括他经常提到《股票魔法师》《笑傲股市》《100倍超级强势股》《以交易为生》《乌合之众》《大癫狂》这些书我都读过，尤其是前三本。

    如果你想把这个模式玩好，这三本书是必读的。我建议从《股魔》读起，先读三遍。

    从博士推荐的书籍和他模式里用到的东西看，非常偏向于行为金融学。




2、上市一年以上

    1、公式

        EXTRS:(C-REF(C,N))/REF(C,N);            // N缺省值为250


    2、通达信 - 菜单 - 功能 - 选股器

        定制选股 -> 其它条件 -> 上市天数 -> 加入条件 -> 上市天数大于360天 -> 选股入板块（上市一年以上）


    3、上市时间筛选 -> 粘贴

        快捷键  =>  60   ->   进入沪深A股板块

            点击键盘右下角向右移动光标的键，直到找到“上市日期”这一列，点击一下，让所有A股 按“上市日期”倒序


        快捷键  =>  34   ->   数据导出

            选中”Excel文件“和“报表中所有数据“

        导出后将一年内上市的股票删除，选中剩下的股票代码，复制到剪切板


        导入“上市一年以上”板块

            通达信 - 菜单 - 工具 — 自定义板块设置 — 上市一年以上 — 从文本中导入 — 粘贴
