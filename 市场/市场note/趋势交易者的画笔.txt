我是如何观察大盘的

    这篇文章的细节，我只会写一次，因为比较啰嗦，未必讨大家会喜欢。但细节和精髓也在里面

    这些都是经验之谈，是我前期通过大量复盘、反思总结出来的。跟操作无关，但却能切切实实提升水平



12月份以来，我一直提倡空仓，防守为主

    年后的下跌在我看来是预期内的

    注意，不是我预测的准，我也没这个能力，这是践行模式的结果，只不过盘面印证了我的预期


为什么提倡空仓？

    空仓的根本原因   ->   市场已经 没有特别适合我的机会

        这个时候要有勇气 空仓或轻仓  ->  不强求模式外的利润

        那些对我来说 更多的是风险

        如果此时不小心消耗了过多的本金  ->  亏钱只是一方面  ->  更严重的是如果后面真的出现机会 -> 你是否还有信心尝试


    模式外的机会

        首先更多的是 风险！！！   ->   其次难度大、很容易犯错，还没有上仓位的勇气      ==>     这种交易，不做也罢

        当然，道理说起来很简单，关键是你能否做到   ->   空仓等待可以说是智慧 -> 但更考验勇气和定力



我是如何观察的？

    前面的文章，我曾多次提及 [市场特性]         // 参见这篇《比交易模式更核心的能力》

        看懂了市场所呈现的特性

            你做股票会变得很轻松，这是我认为走向独立交易的第一步

            可以不再依赖别人的分析，有了更明确的方向性，知道哪里有机会，哪里有风险，机会的性价比又如何


那么市场这么大，我们怎么看？

    1、看涨的思路：
        1、涨的最好的板块
        2、涨的最好的板块的位置（高位、低位）
        3、涨的最好的板块容量大小
        4、涨的最好的板块效应如何
        5、板块内部主导资金（机构、游资）
        6、炒作题材的逻辑 （概念、业绩）


    2、看跌的思路：
        1、跌的最凶的方向
        2、跌的最凶的板块位置（高位、低位）
        3、跌的最凶的板块影响范围
        4、跌的最凶的方向与指数关系
        5、跌的最凶的板块主导资金（机构、游资）
        6、杀跌的逻辑（长期、短期）




做好了这些，基本对市场影响最大的因素搞清楚了  ==>  也就是 重视两极（多/空）影响



为什么要这么做？有什么用？能得到什么？

    1、看板块位置

        可以知道 这个机会是新还是旧

            新机会 ->  要放入自选观察     ->      做到先知先觉，第一时间跟踪到

            老机会 ->  要看此时板块内部 股价的 走势如何，性价比如何，是否还值得参与     ->   做到主动规避风险，不满足模式的坚决不做



    2、看板块容量大小

        可以知道 这个方向资金容纳情况

            是否可以满足 各类资金都来介入     ->      一旦炒作，资金的介入 深度和范围 如何


    3、看板块效应

        可以确定 炒作的强度，也就是溢价         // 这个看 [涨停数量] 就可以


    4、看主导资金

        机构  ->  参与的越多当然越好   ->   代表了 大资金的态度

        游资  ->  比如纯题材类的  ->  就是游资主导的   ==>   他们的 影响力有限  ->  不会 对整个大盘造成太大影响

                    // 炒小炒差 - 妖股接力（炒概念-无业绩支撑） - 击鼓传花


    5、看板块与指数的关系

        可以确定 这个板块的涨跌  ->  能否影响大盘整体情绪

            比如前段时间 医药和新能源 一起杀跌  ->  对创业板指数影响很大  ->  继而又造成了 整个盘面的恐慌情绪


        不要以为指数不重要

            它的走势   ->   可以判断 整个盘面的情绪

            如果想要有行情（板块）   ->   指数（大盘）就一定不能杀跌




我简单写了几点思路，看涨和看跌其实道理一样，反过来想就可以了

    关键还是要亲自实战，别看我上面写的啰嗦，只要熟练了其实很轻松


    有时候简单看一下，就知道有没有机会值得观察跟踪

        如果没有必要，那后面的操作就不需要了，一开始就定性了

        我只需要等待符合模式标准的机会就可以，其它的涨跌我只看戏



那文章开头的问题，为什么提倡空仓？怎么理解呢？

    1、我观察到 从12月份开始 市面上已经完全换了一种炒作风格   ->   就是 炒小炒差

        这些基本都是 游资主导的   ->   暴涨暴跌  ->  风险是肯定大于机会的

        虽然我前期也选到了雅本化学、翰宇药业、迦南科技、左江科技这类机会，他们的股价走势是满足模式标准的

        但当时在外面游玩，错过了我模式最佳的上车机会，所以还是放弃了


    2、年前12月1号那天我记得美股大跌了，但是A股确是低开高走的，很不错

        如果A股真的能独立，后面要持续涨才对

        但12月2号，反而跌了不少

        上证指数看不出来，创业板可能更明显一些，也就是说当时市场有明显杀跌的方向，这不满足我的预期


    3、通过最近的炒作风格，我发现机构明显不干活了

        典型就是 -> 龙虎榜上的 机构出现次数 越来越少，金额 也是越来越少


        也就是说，大资金这时候不积极了

            他们没有主动站出来引领市场，反而游资炒的风生水起


        一般一个题材出现

            机构参与的越多  ->  这个题材持续性会越好 -> 对大盘的积极影响也就越好

            像 元宇宙这类  ->  完全不是机构主导的


            新能源、医药板块

                在12月初就已经有回调的迹象了

                盘面远远弱于我的预期，我更加谨慎了一些



    经过上面的分析

        我基本确定了这段时间的策略  ->  就是 谨慎小心，空仓观望


        至于后面能躲避过大跌，这我无法预测

        只能说，我的模式不会轻易经历这种下跌

        回撤也就自然的控制住了


    至于何时开始做多

        当然我只做符合模式标准的机会

        见前面一篇文章《我在等什么》

        不急的，只要市场还有炒作的东西，理论上都会有赚钱的机会，耐心等待

板块炒作


对于一个新的方向，我是这么计划的：

    持续的跟踪：


    1、看整体

        观察这个 板块整体 的发育情况

            是否会 继续扩大 炒作范围

            是否有 持续的资金 参与进来    ->  比如 龙虎榜 越来越多


    2、看强势股

        看板块内部 强势股 的走势

            1、看前高（压力位）               // 率先拿下高地的  ->  即为猛将（观察它 -> 加入它）

                前面 是否有炒作过 的痕迹

                    也就是 前期的高点（压力位）在哪里

                对于一个新题材

                    一年新高（突破前高 - 压力位）    ->   肯定要比半年新高更好一些

                    至少说明，有可能会开启一波新的上涨，股价上方的压力比较小


            2、看量价（放量上涨，缩量回调）      // 百万雄师过大江   ->   打过长江去，解放全中国

                看板块内部强势股后续的 量价走势


    3、找买点

        （突破、回踩）如果第一时间没有参与，可以等后面缩量回踩的时候，寻找一个最佳平衡点


    4、去弱留强

        跟踪的这些个股，可能后续走势会不同   ->  这是一个淘汰的过程

        只留下走势符合预期的  ->   参与难度较小的，止损空间较小的

        至于市场   ->   给不给我们机会   ->   有没有走成 我们预期中的样子   ->   我们无法预知






针对题材炒作规律，这里主要分两点：


    1、第一波 只炒作了三五天  ==>  就停车休息           // 停车观望 - 这时候仔细跟踪 -> 看是否会走趋势


        题材（整体板块）没有继续发酵  ->  只有 一小部分个股（强势股） 率先走强

            市场对这个方向  ->  有分歧 或 还没有更大的消息刺激到它

            然后股价调整，但 [强势股] -> 不会跌的很深


        后续再次炒作的时候   ->  资金也不知道 第一时间买什么

            但 前期这些强势票 会先受到 资金的青睐   ->  走出第二波主升浪行情   ->  继而刺激到 更多的资金 挖掘板块内部 其它个股

            这里的合力会更大一些 -> 形成强势的板块效应

            后续股价的每一次调整，都是对强势股的洗礼


    2、第一波炒作  ==>  就持续的拉升（妖股）    ->  比如 连板上去的  ->  后来又 无量加速板

        这种很容易 一波流  ->  股价短期内严重透支了空间

            但也会有个别的大妖股，每次分歧后，可以不断的续命

            后续的回调要么很暴力，要么止跌后，反弹无力，更别提反转继续新高了



    -----------------------------------------------------------------
    很明显，第一种才是趋势的样子

        尤其是针对机构主导的行情   ->   会有[慢牛股]出现

        虽然看每一波拉升都不会很高，但时间长了，涨幅惊人


    同理

        昨天出现的预制菜、智能汽车板块，也可以用相同的思路来做应对

    其实炒作

        无非就是那么两三种样子，做的多了自然就很轻松了
