再谈走势与买点


1、结构紧凑的调整   ==>     均线粘合   ->   筹码交换（主力缓慢收集筹码 - 时间换空间）


    绝大部分走长期趋势的牛股，前期都会经历这种结构紧凑的调整。有的是一年，有的是几个月。

    调整的上下波动在一个较小范围内，均线的样子就是粘合，没有明显的发散。可以理解为，这是一个筹码交换的过程。为后面的趋势行情打下了一个坚实的基础。

    在这个调整阶段，大盘肯定会经历了上下波动，但这种股票，没有被市场资金发掘，主力可以很好的控盘收集筹码。

    是趋势酝酿阶段最重要的一步，也是必经的一步。


    前期整理的越充分，后期的空间也就越大。俗话说横有多长，竖有多高！


2、均线粘合


    1、均线粘合

        你可以随便翻开一只趋势牛股，必定会经历均线粘合的过程

        一般发生在趋势酝酿阶段


    2、多头排列

        在经过最后一次下跌后，均线终于被整理的有序

        短、中、长 期均线依次多头排列

        这往往发生在第一波主升浪时期

        这是因为市场第一次炒作它，刚从震荡阶段走出来，所以均线有序排列


    3、上下交错

        很多趋势股，到最后阶段，均线不在有序排列，而是非常纠结，上下交错


    4、空头排列




3、良好的量价关系


    上涨放量、下跌缩量

        趋势酝酿的最后一段时间，一般会出现明显的上涨放量，下跌缩量现象。这就是红肥绿瘦。

        最后的那一段调整，成交量突然极度萎缩。等调整结束第一根放量的反包阳线，就是最佳的买点。


4、走势流畅

    很多人忽视了这个关键点，这点直接决定了你能否等来一个正确的舒服的买点，能否稳健持股，最后又能带走多少利润。



    什么是流畅的走势

        这只票虽然经过几次起起伏伏，但是中间的过程，走的非常顺

            没有剧烈的    震动

            更没有太多的  上下影线

        在这种走势上参与，成功的概率将大大增加，而且最后你能带走的利润，也是最多的

        你不会被各种奇怪的震荡欺骗，不会被震荡洗出来



    再看一下那种不流畅的走势

        频繁上下起伏（周线不平滑）       ==>  上上下下   ->   涨跌 无规律、无规则

        频繁  上下影线                ==>  盘中 急剧拉升、下跌   ->   拼盘被骗


        ----------------------------------------
        有些票，最后可能涨了不少，但是持股过程非常难受

        你很容易在这种上下震荡中犯错，最后赚不到什么钱，而且也不敢上仓位




走势相关的基本写完了，这里其实涉及到很多选股的思路。也算基础里非常重要的部分。更细节的东西，在选股里会补充说明。

选股


永远只关注市场上最强势的那批股票

    这是市场合力选择，是主力资金的选择

    是它们引领行情发展，同样也是获得超额收益的源泉


    这句话不仅仅是选股思路，更是交易的核心理念

        先选对方向，操作才会变得简单，技术更是辅助


    市场在变，公司在变，炒作的逻辑也在变

        但是牛股的走势规律不变，人心不变，情绪不变


    这种强势股的思维

        可能大部分人都理解不了，觉得很难，且风险较大

        或者说根本就没有这种意识

        他们喜欢那种低位的，觉得安全




我直接用前面讲的基础知识，拿最近操作过的几只票来解释一下，为什么当时选择它，如何分析，买点在什么位置。用一个套路，不断重复的去应对绝大部分牛股。

    1、中远海控

        中远海控第一波从底部起来，其实已经涨了2倍左右。最后还出现了一波放量的加速上涨，这已经是一只牛股了。


        在第一波主升结束后的回调，只跌了大概15%左右

            对于一只涨了近2倍的票，仅仅跌了15%就出现企稳迹象，就已经很强势了。

            而且回调非常健康，没有明显出货迹象，成交量持续的萎缩，说明卖盘非常少。


        等持续回调后，出现第一根放量的反包k线时，就是最佳买点

            这里很可能是调整结束的信号，在这里参与，是盈亏比最佳的位置。

            至于后面能涨多少，又会出现什么样的走势，这在当时无法预测，也不需要预测。

            即便错了，这里的止损也是非常小的。与潜在的回报比，完全值得一试。



        中远海控的第二波，同样可以用前面的方法继续分析一次。

            第一波回调的幅度还是很大的，但经过几次震荡，反而走出了向上的迹象。

            再次涨到了前高的位置，并出现了放巨量的阴线，看起来就是顶部。

            这里肯定不会去赌突破。因为短期涨的太快了。这里也是一个重要的心理压力位置。

            耐心等待回调企稳后，出现了开始转折的买点。此时的均线也起到了支撑的作用，后面随着股价逐步抬升。



    2、科沃斯

        科沃斯，去年的大牛股。

        可以看它的走势，典型的趋势牛股，而且每一次回调，都是一次绝佳的买入机会。

        分析方法跟海控的一样。



    3、大豪科技

        大豪科技前期出现了一个非常好的结构紧凑的整理，而且股价缓慢抬升，配合当时白酒的炒作氛围，这种走势，大概率会走牛。



    4、瑞芯微

        股价经过长期的震荡整理，开始放量站上所有均线，此时出现横盘震荡走势

            并且下跌都是严重缩量的，代表卖盘很少

            均线也从粘合，慢慢向上发散，依次有序排列

            股价回调时，均线也起到了支撑作用


        一波行情，都是从均线密集开始的，等到均线严重乖离结束

            如果一只股票，未来会上涨5倍，10倍，那肯定是先从1倍涨起来的

            我们不需要去挖掘这种牛股，市场会帮你选择

            你只需要在已经走牛，或者刚刚走牛的股票里等待机会就可以

            这充分体现了强势股的交易理念




    结合这篇的分析，再加上之前几篇的内容，模式内技术类的东西已经讲得差不多了。其实已经具备了参与市场的条件。

        我并没有讲那些花里胡哨的技术指标

            前期我也研究过，macd kdj rsi boll等等，

            但经过我长期的复盘和实战，最终留下的就是 k线、成交量、均线 三者

            其它的那些指标，都是这些变换出来的，大可不必花精力去学


        对市场的理解，对人性的理解，对情绪的掌控等等，这些才是交易里更重要的东西

            技术，仅仅是辅助。千万不要陷入技术的迷途里



        而强势股的思维，会让很多东西都简化掉，甚至都不用考虑

            所以我一直说，别着急，一切都会变得简单

何为最佳走势



都说炒股要耐心等待机会，那我们应该把耐心用在什么地方，又在等待什么，真正的机会又是什么样子的

    有了前面对走势的理解，今天说一下，我认为最完美的走势

    这篇并没有新的知识点，只不过把内容更加提纯了，但却是我认为非常重要的一篇


    所有的牛股

        都会有一个共同的规律   ==>   就是不断的突破新高   ->   翻倍再翻倍


        我们需要做的其实很简单

            把这些走牛的股票收集起来  ->  耐心等待他们回调的时候 -> 观察他们的量价与走势  ->  判断值不值得参与就可以



先看几张牛股的走势图：

    2019年的立讯精密

        1、先涨了一波

        2、回调又再次涨到前高

        3、走不断创新高的趋势


        每次回调都有不断上车的机会



    2020年的立讯精密

        1、经过较深回调后，又再次涨到前高

        2、再次回调

        3、开始突破走加速趋势



    2020年的山西汾酒，周线图

        1、先从28涨到90，翻了2倍多        // 已经是牛股了

        2、进入结构紧凑的中期调整

        3、再次突破后，走趋势上涨

            在90的基础上，又涨到了500



    2019年的圣邦股份

        1、在前面新高的基础上，经过了一波回调

        2、涨到前高位置附近

        3、进入结构紧凑的中期调整

        4、再次突破，走主升浪趋势

        1、先涨到前高附近

        2、经过回调又突破了前高

        3、经过最后一个回调后，开始了主升趋势

    股价翻倍再翻倍

    是市面上最强势的那一批

    1、股价先走强一波

        可能会翻倍，或者突破长期的前期高点

    2、进入结构紧凑的中期调整

    3、股价再次涨到前面的高点附近

    4、再次经过一阵短暂的回调后，突破走主升浪

指数、板块、个股运行规律


行情都是从大跌中酝酿的

    只要指数没有主跌浪，或者大盘没有系统性风险，只需要关注强势板块的强势个股即可

    说的傻瓜一点，就是跌多了涨，涨多了跌



    那应该何时重点关注，何时小心谨慎呢？

        我画了一张图，描述了 板块与个股 的运行规律

        这里面还包含了 情绪的周期




1、本轮指数的上涨

    由几个强势板块拉动起来的

    此时到了该回调的阶段，开始一轮下跌，并带动指数一起下跌

    情绪逐渐打到冰点


2、某些板块提前指数下跌

    在这轮下跌中明显跌的比较少，显示出了比大盘强的状态

    尤其是一些强势个股，跌幅更少，而且已经在慢慢磨底



3、大盘指数开始企稳，不再创新低

    震荡盘整时，有一些板块就悄悄的爬升

    此时的指数，看起来还是比较羸弱，没有要进攻的欲望


4、强势板块里的强势个股，已经慢慢悠悠爬升了一段，快要到前高附近了

    而且呈现明显的   ->   上涨放量，下跌缩量



5、指数经过一段时间的震荡，终于从泥潭里开始往外爬

    此时的强势板块 由一些个股带动 也涨了起来

    比指数有更明显的上涨欲望


6、一批强势个股在前期高点附近徘徊，震荡整理，但没有出现明显的放量下跌

    终于在最后一次回调整理后，选择了强势加速突破，走主升浪


7、此时被这些强势板块的带动，大盘整理也情绪开始恢复，热度慢慢抬升

    板块轮动此起彼伏，就连一些没有明显利好或者逻辑的个股都开始上涨了

    大盘情绪从冰点到恢复阶段，赚钱往往是最容易的

    恢复以后，需要不断有板块炒作来给大盘续命，带着情绪继续走向更高潮



8、经过一轮的炒作后，指数又来到了高位

    此时情绪有点疲惫了，有了回调的需求

    开始新一轮的下跌

    但往往下跌很少一蹴而就，不断的给你希望又让你失望




描述的可能不够细致，但核心就是这些

    只要掌握了这种 大盘与板块、个股 的运行规律，就知道当下的机会与风险了


    可以复盘一下今年从1月开始，到现阶段，所有被炒作过的板块、个股

        比如 4月份的医美、5月的白酒和锂电、6月的半导体

        这些强势的板块有些已经涨了几倍  ->  但大盘指数其实没怎么涨


        所以还是回到最开始的交易理念

            在指数没有主跌和风险时  ->  始终要关注 最强的那一批股票



那么思考一个问题，市场当下，还能炒作什么呢？

    哪里还可以为大盘续命呢？小弟不知啊！

怎么卖？！


这两天盘面的走势，大部分要么拍大腿，要么拍断大腿。反复被摩擦

    在你认为的买点买入之后，股价没有迅速脱离成本区，反而跌了，那说明买点还是不对

    这个时候不要有侥幸心理，先出来观望

        而且也跌破了这里构建的小平台，这个时候就是关键点

        当时我在分享这只票时，就说了， 越接近缺口是第一买点

        也是盈亏比较高的位置，如果这里的买点成立，那就可以守着这个最低点，上面空间还是挺大的



        从这个票的走势来反思，还是对选股的要求更高一些。不要随意预测股价未来的走势。



    康德莱

        当时提示位置是1

        在前面几天股价企稳

        那么交易就简单了

            2这里买入之后，守着前面几天的低点，只要股价没有跌破，就可以拿着

            但我的思考是，股价短期走得太快，太强，属于趋势里最强的一种走法，但这不可持续

    石大胜华

        往往股价最后还有一段加速的过程   ==>   就比如这几天，明显加速冲顶了   ->   属于情绪的高点 -> 卖！！！

    股价走出了明显的顶部迹象，比如高位震荡，多日不在创新高了

风险也是周期的一部分

    这个时候只有卖点，没有买点



如果目前的盘面，你看不懂   ==>   就不要随意出手  ->  往往这个时候也是 亏钱最凶残的时候

    情绪到了高潮以后，从一个极端瞬间到另一个极端

    等风险彻底释放后，再耐心找机会



这种情况在 科技类 上最容易出现

    每一次半导体或者科技股走的都是尖顶

    就是因为情绪演绎到了极致





任何一个板块或者个股走牛

    2、直到均线严重发散结束

    3、最后又会回归到粘合状态

        也许是通过横盘震荡，也许是通过下跌来实现

        这是永远不变的规律




即便是十倍股，百倍股

    主升浪就是最肥美的阶段，不要贪

就像医药板块的这些个股

我是如何观察大盘的

    这篇文章的细节，我只会写一次，因为比较啰嗦，未必讨大家会喜欢。但细节和精髓也在里面

    年后的下跌在我看来是预期内的

    注意，不是我预测的准，我也没这个能力，这是践行模式的结果，只不过盘面印证了我的预期


为什么提倡空仓？

    空仓的根本原因   ->   市场已经 没有特别适合我的机会

        这个时候要有勇气 空仓或轻仓  ->  不强求模式外的利润

        那些对我来说 更多的是风险

        如果此时不小心消耗了过多的本金  ->  亏钱只是一方面  ->  更严重的是如果后面真的出现机会 -> 你是否还有信心尝试


    模式外的机会

        首先更多的是 风险！！！   ->   其次难度大、很容易犯错，还没有上仓位的勇气      ==>     这种交易，不做也罢

        当然，道理说起来很简单，关键是你能否做到   ->   空仓等待可以说是智慧 -> 但更考验勇气和定力



我是如何观察的？

    前面的文章，我曾多次提及 [市场特性]         // 参见这篇《比交易模式更核心的能力》

        看懂了市场所呈现的特性

            你做股票会变得很轻松，这是我认为走向独立交易的第一步

            可以不再依赖别人的分析，有了更明确的方向性，知道哪里有机会，哪里有风险，机会的性价比又如何


那么市场这么大，我们怎么看？

    1、看涨的思路：
        1、涨的最好的板块
        2、涨的最好的板块的位置（高位、低位）
        3、涨的最好的板块容量大小
        4、涨的最好的板块效应如何
        5、板块内部主导资金（机构、游资）
        6、炒作题材的逻辑 （概念、业绩）


    2、看跌的思路：
        1、跌的最凶的板块
        2、跌的最凶的板块位置（高位、低位）
        3、跌的最凶的板块影响范围
        4、跌的最凶的方向与指数关系
        5、跌的最凶的板块主导资金（机构、游资）
        6、杀跌的逻辑（长期、短期）




做好了这些，基本对市场影响最大的因素搞清楚了  ==>  也就是 重视两极（多/空）影响



为什么要这么做？有什么用？能得到什么？

    1、看板块位置

        可以知道 这个机会是新还是旧

            新机会 ->  要放入自选观察     ->      做到先知先觉，第一时间跟踪到

            老机会 ->  要看此时板块内部 股价的 走势如何，性价比如何，是否还值得参与     ->   做到主动规避风险，不满足模式的坚决不做



    2、看板块容量大小

        可以知道 这个方向资金容纳情况

            是否可以满足 各类资金都来介入     ->      一旦炒作，资金的介入 深度和范围 如何


    3、看板块效应

        可以确定 炒作的强度，也就是溢价         // 这个看 [涨停数量] 就可以


    4、看主导资金

        机构  ->  参与的越多当然越好   ->   代表了 大资金的态度

        游资  ->  比如纯题材类的  ->  就是游资主导的   ==>   他们的 影响力有限  ->  不会 对整个大盘造成太大影响

                    // 炒小炒差 - 妖股接力（炒概念-无业绩支撑） - 击鼓传花


    5、看板块与指数的关系

        可以确定 这个板块的涨跌  ->  能否影响大盘整体情绪

            比如前段时间 医药和新能源 一起杀跌  ->  对创业板指数影响很大  ->  继而又造成了 整个盘面的恐慌情绪


        不要以为指数不重要

            它的走势   ->   可以判断 整个盘面的情绪

            如果想要有行情（板块）   ->   指数（大盘）就一定不能杀跌

板块炒作


对于一个新的方向，我是这么计划的：

    持续的跟踪：


    1、看整体

        观察这个 板块整体 的发育情况

            是否会 继续扩大 炒作范围

            是否有 持续的资金 参与进来    ->  比如 龙虎榜 越来越多


    2、看强势股

        看板块内部 强势股 的走势

            1、看前高（压力位）               // 率先拿下高地的  ->  即为猛将（观察它 -> 加入它）

                前面 是否有炒作过 的痕迹

                    也就是 前期的高点（压力位）在哪里

                对于一个新题材

                    一年新高（突破前高 - 压力位）    ->   肯定要比半年新高更好一些

                    至少说明，有可能会开启一波新的上涨，股价上方的压力比较小


            2、看量价（放量上涨，缩量回调）      // 百万雄师过大江   ->   打过长江去，解放全中国

                看板块内部强势股后续的 量价走势


    3、找买点

        （突破、回踩）如果第一时间没有参与，可以等后面缩量回踩的时候，寻找一个最佳平衡点


    4、去弱留强

        跟踪的这些个股，可能后续走势会不同   ->  这是一个淘汰的过程

        只留下走势符合预期的  ->   参与难度较小的，止损空间较小的

        至于市场   ->   给不给我们机会   ->   有没有走成 我们预期中的样子   ->   我们无法预知






针对题材炒作规律，这里主要分两点：


    1、第一波 只炒作了三五天  ==>  就停车休息           // 停车观望 - 这时候仔细跟踪 -> 看是否会走趋势


        题材（整体板块）没有继续发酵  ->  只有 一小部分个股（强势股） 率先走强

            市场对这个方向  ->  有分歧 或 还没有更大的消息刺激到它

            然后股价调整，但 [强势股] -> 不会跌的很深


        后续再次炒作的时候   ->  资金也不知道 第一时间买什么

            但 前期这些强势票 会先受到 资金的青睐   ->  走出第二波主升浪行情   ->  继而刺激到 更多的资金 挖掘板块内部 其它个股

            这里的合力会更大一些 -> 形成强势的板块效应

            后续股价的每一次调整，都是对强势股的洗礼


    2、第一波炒作  ==>  就持续的拉升（妖股）    ->  比如 连板上去的  ->  后来又 无量加速板

        这种很容易 一波流  ->  股价短期内严重透支了空间

            但也会有个别的大妖股，每次分歧后，可以不断的续命

            后续的回调要么很暴力，要么止跌后，反弹无力，更别提反转继续新高了



    -----------------------------------------------------------------
    很明显，第一种才是趋势的样子

        尤其是针对机构主导的行情   ->   会有[慢牛股]出现

        虽然看每一波拉升都不会很高，但时间长了，涨幅惊人


    同理

        昨天出现的预制菜、智能汽车板块，也可以用相同的思路来做应对

    其实炒作

        无非就是那么两三种样子，做的多了自然就很轻松了
