板块效应 的真正含义

    板块是资金划分的，资金在发动一个板块的时候，拉动了谁，谁就是这个板块的。

        这看上去是一句废话，鸡生蛋还是蛋生鸡

        但很多人其实不会判断一个股是不是属于某板块，而是高度依赖一些软件里面的划分


    我之前遇到很多和我争论某个股不是某板块…这个具体怎么区分呢？

        就是板块集体动的时候它也动，分时的节点比较重合

        一次可能是巧合，但是板块的异动一天可能两次三次，一波行情可能好几天，全部重合那肯定不是巧合了

交易核心思想                      // 思路正确是成功的第一步  ->  路线错误，知识越多越没用

    预判与跟随

        1、对预判的预判

            金融投资如同选美

                别猜你认为最漂亮的美女能够拿冠军，而应该猜大家会选哪个美女做冠军


            我记得我聊过有一年我做新易盛的心得

                财报大增不一定涨，财报超预期不一定涨，财报真的 超了机构的预期 -> 才会涨

                我是从那时候开始发现，人们交易的不是股票，不是公司，而是对预判的预判



        2、所有人的想法

            我们经常会下意识的按照 “我的想法就是所有人的想法” 来操作


            20世纪20年代，心理学家让-皮亚杰注意到

                年幼的孩子常常会把自己对事物的认识与该事物的本质混淆起来，以为其他人看到的一定跟自己看到的一样

            随着他们越来越成熟

                孩子们的世界观就由唯实主义转变成唯心主义

                渐渐认识到自己的想法只是一种观点，并不一定是客观事实

                所以，两个人对同样的事情可能会有不同的观点



            即使成年之后

                我们的认知过程最初还要经历 唯实主义思维的阶段

                所以，我们一不小心还是会栽了  =>  把我认为 当成 所有人都认为



            所谓的 资金  ->  就是“所有人”

                所以，市场爸爸 是我们了解所有人的想法的唯一方式


        3、市场爸爸说了什么

            市场有千千万万的主体组成

                大的公募，各种机构，小的大小散户，游资，各有各的想法和风格，可以说是一片喧嚣

                在通过无数的买卖单的沟通和斗争过程中，分歧被统一，喧嚣中形成一个主流，这个主流就是大多数人的想法

                少数否定市场、把自己当做永远正确的人，依然是市场的杂音

            那些努力与市场沟通的人

                逐渐掌握了市场的语言，在市场里如鱼得水

                而固执得认为自己才是正确的，放弃与别人沟通，或无法掌握市场语言的，就一点点亏钱，逐渐淘汰


            一个投资者，要么学会这门语言，要么被淘汰


        4、物理学还是生物学

            资本市场是一个生态系统

                是一个大鱼吃小鱼、小鱼吃虾米的环境

                你能适应这个生态你就能存活，有“大哥”且没天敌就能壮大，如果不能适应就只能被淘汰


            在投资领域对比一下物理学方向与生物学方向的方法论，其实一个是本质主义，一个是群体思想

                从物理学的方法里，你会发现你根本无法预测

                    在一张巨大的网络里，一层加一层，参数指数级增加永无尽头。没有这样的方程，也没有这样的确定性存在。

                基于生物学的方法论  ->  其实就是 应对

                    恩斯特-迈尔很无奈的说，生物学中的概括几乎完全是几率性的。生物学中只有一条普遍定律，那就是一切生物学定律都有例外。

                    生物学的方法，例外和新变化 -> 往往是市场告诉你的宝贵方向



            本质主义  ->  是直觉上 非常正确的

                研报一出，你就感觉专业；逻辑一出，你就十分信服，仿佛有一种毋庸置疑的魅力，很容易就让你想买入。


            而无法预测，着重应对  ->  仿佛是一种低能的表现

                事实上，预测  ->  是一种命中率很低，至少比你以为的靠谱程度 要低的路径

                而 应对  ->  却是在丛林法则下 生存的必备技能

                    听懂市场的语言  ->  做出相应的应对

                    让大资金的惯性  ->  为你的灵活买单


            生物学的方法 -> 实际上并不是无法预测

                春江水暖鸭先知

                资金的动作  ->  往往比你能知道的 消息和逻辑 快一个月左右
