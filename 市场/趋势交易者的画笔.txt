3、RPS

    这个指标是欧奈尔提出来的


    RPS（股价相对强度）

        就是在一段时间内 -> 股价的涨幅 在整个市场的排名   =>   数值越大，排名越高



    为啥这个指标那么重要？


        因为这是市场的选择   ->   是市场的合力造就的

            从  天时、地利、人和  三方面都满足了要求


        一个板块或者一只股票  ->  不会无缘无故的涨

            首先 大盘整体环境没有问题、逻辑也得到了认可、最后资金达成了共识

            这个时候，也是这只股票 最有确定性的时候

            没错，一只股票最有确定性的时候，就是涨起来的时候



        股价在底部区域时 -> 有很多不确定因素

            比如时间上未知，你无法预测市场什么时候再次挖掘到它，在底部盘整几个月，几年很常见

            而且有些票，越跌越危险，时常伴有各种利空出现，利好也是利空

            但涨起来就不一样了   ->   利空 也被解释成利好             // 本质上 -> 是筹码 转移到了主力手中


        你细品！！！




4、结构紧凑的中期调整

    这句话以前博士经常提到，我也是一脸懵逼，到底何为结构紧凑的中期调整呢？

        见图


    简单的说

        就是一段波动较小的、横盘、缩量整理走势

        这种走势一旦开始放量上涨，就意味着可能一段新的行情将要展开


    这一条非常非常重要

        是一只股票走牛的基础条件


    那些 震荡幅度很大的   =>   就很难走牛

        也许里面的   ->   资金掌控能力不足，或者某些东西存在瑕疵

6、正确的买点


    坚持正确的买点

        买入之后股价迅速脱离成本区，没有止损的机会。

        即便止损，也会控制在最小范围。



    这句话是经典中的经典

        博士模式里大部分内容，都是与正确的买点有关

        这也是我从他那里学到的最深刻、最核心的东西



    一个好的买点

        不仅决定你的收益如何，更能决定你持股的心态如何

        拿得住，才能赚得多

        如何买，我前面有很多文章都提了，可以反复阅读



    举两个比较标准的买点：


        第一买点   ->   突破买

            注意一点，前面的走势没有较大的波动，股价没有被炒作过，算是一个新的机会，这样空间才会打开


        第二买点   ->   回踩买

            股价回调企稳时  ->  可以二次上车

                一定要深刻认识，回踩买   =>   买的 不是回踩 -> 而是转折

                    这个转折买点一旦成立 -> 股价就向上运动，脱离成本区

                    即便错了，这里也是止损点，止损空间较小



            但要注意：

                随着时间和趋势的推移

                    越早的回踩   ->   越安全

                    越到后面     ->   越危险


                也就是一波新的炒作行情

                    在股价第一次回调时   ->   是最具备性价比的

                    越到后面抛压会越大   ->   最终有一次回踩会失败
