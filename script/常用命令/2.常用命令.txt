说明：安装linux时，创建一个bebop用户，然后使用root用户登陆系统

1.进入到用户根目录
cd ~ 或 cd

2.查看当前所在目录
pwd

3.进入到bebop用户根目录
cd ~bebopze

4.返回到原来目录
cd -

5.返回到上一级目录
cd ..

6.查看bebop用户根目录下的所有文件
ls -la

7.在根目录下创建一个bebop的文件夹
mkdir /bebopze

8.在/bebop目录下创建src和WebRoot两个文件夹
分别创建：mkdir /bebopze/src
		  mkdir /bebopze/WebRoot
同时创建：mkdir /bebopze/{src,WebRoot}

进入到/bebop目录，在该目录下创建.classpath和README文件
分别创建：touch .classpath
		  touch README
同时创建：touch {.classpath,README}

查看/bebop目录下面的所有文件
ls -la

在/bebop目录下面创建一个test.txt文件,同时写入内容"this is test"
echo "this is test" > test.txt

查看一下test.txt的内容
cat test.txt
more test.txt
less test.txt

向README文件追加写入"please read me first"
echo "please read me first" >> README

将test.txt的内容追加到README文件中
cat test.txt >> README

拷贝/bebop目录下的所有文件到/bebopze-bak
cp -r /bebopze /bebopze-bak

进入到/bebopze-bak目录，将test.txt移动到src目录下，并修改文件名为Student.java
mv test.txt src/Student.java

在src目录下创建一个struts.xml
> struts.xml

删除所有的xml类型的文件
rm -rf *.xml

删除/bebopze-bak目录和下面的所有文件
rm -rf /bebopze-bak

返回到/bebop目录，查看一下README文件有多单词，多少个少行
wc -w README
wc -l README

返回到根目录，将/bebop目录先打包，再用gzip压缩
分步完成：tar -cvf bebopze.tar bebopze
		  gzip bebopze.tar
一步完成：tar -zcvf bebopze.tar.gz bebopze
		  
将其解压缩，再取消打包
分步完成：gzip -d bebopze.tar.gz 或 gunzip bebopze.tar.gz
一步完成：tar -zxvf bebopze.tar.gz

将/bebop目录先打包，同时用bzip2压缩，并保存到/tmp目录下
tar -jcvf /tmp/bebopze.tar.bz2 bebopze

将/tmp/bebopze.tar.bz2解压到/usr目录下面
tar -jxvf bebopze.tar.bz2 -C /usr/
